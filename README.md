# zyx-china.github.io

基于 [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) 主题的个人主页，部署于 GitHub Pages。

## 本地运行

```bash
bundle install
bundle exec jekyll serve
```

## 目录结构

```
├── _config.yml          # 站点配置
├── _data/navigation.yml # 导航菜单
├── _pages/              # 静态页面
│   ├── about.md         # 关于
│   ├── cv.md            # 个人简历
│   └── projects.md      # 项目经历
├── index.md             # 首页
└── Gemfile              # Ruby 依赖
```
