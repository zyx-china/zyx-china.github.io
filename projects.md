## 量化策略研究

### 中高频宏观因子行业景气轮动策略 — 宏观周度截面因子
*东吴证券 · 金融工程组*

- 开发宏观指标与资产代理回归筛选框架，通过月频重采样、插值及季节调整等技术将低频宏观变量高频化
- 利用 Bootstrap 和 HAC/Newey-West 方法检验 β 稳定性，构建可交易的宏观因子库
- 设计七大类宏观因子的日频至周频特征工程并预测行业盈利与估值变化
- 构建景气与估值共振信号用于行业轮动策略，回测年化收益 **13.36%**，夏普比率 **0.41**（基准 8.28%、0.22）

### 基于 TIPS 的贵金属预期收益择时因子
*东吴证券 · 金融工程组*

- 采用月度数据扩展窗口滚动 OLS 估计 TIPS 对贵金属远期回报的斜率
- 通胀调整后将月度预测映射为日度信号供回测
- 设计多套策略：简单阈值、分批建仓（1/12持有12个月）、12 slot 轮动、滚动12期均值仓位及其 long-only 变体
- 将检验与策略复制到上期所白银期货

### 港股多维视角风格轮动定量模型
*东吴证券 · 金融工程组*

- 搭建港股风格轮动量化研究框架，保证因子计算方式与回测的可扩展性
- 构建若干月度宏观择时因子与月度轮动组合策略，预期进一步引入周度量价因子

## Nexargus — 金融分析 Agent
*2026年04月 - 2026年05月 · [github.com/zyx-china/nexargus](https://github.com/zyx-china/nexargus)*

- 从零构建的多层架构金融分析智能体系统，实现 **15 层 Agentic 能力**（含 5 种算法优化策略），具备规划、反思、辩论、元认知等高级认知能力
- **TTT 测试时训练**：临时领域知识注入，用完即弃
- **GRPO 群组相对策略优化**：N样本 → 群组排序 → Best-of-N 选择或 Top-K 融合
- **Long-Horizon 长程任务管理**：检查点快照 + 子目标验证 + 自动回溯防漂移
- **Self-Consistency 自洽采样**：多推理链投票共识
- 完整实现"思考-行动-观察-反思-辩论-优化"多阶段推理循环：LLM 通过 Function Calling 自主调度 19 个工具获取实时金融数据，分析结果经置信度校准后进入自我反思审核，多个不同人格子代理可并行或辩论式协作
