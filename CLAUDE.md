# zyx-china.github.io

GitHub 个人主页项目。用户 zyx-china 的个人主页，仓库 github.com/zyx-china/zyx-china.github.io。

## 技术栈
- Jekyll + Minimal Mistakes 主题
- GitHub Pages 自动部署

## 文件结构
- `_config.yml` — 站点配置
- `_data/navigation.yml` — 导航菜单
- `_pages/cv.md` — 个人简历
- `_pages/projects.md` — 项目经历
- `_pages/about.md` — 关于本站
- `index.md` — 首页
- `assets/images/bio-photo.jpg` — 头像

## 更新流程
用户发送更新内容时，直接编辑对应文件，然后 git commit 并 push。
